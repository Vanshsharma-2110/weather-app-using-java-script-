# 🌤️ Weather Dashboard

A sleek, responsive weather dashboard built with vanilla HTML, CSS, and JavaScript — powered by the WeatherAPI.com API.

## ✨ Features

- 🔍 Search any city worldwide
- 🌡️ Live current temperature, feels-like, wind speed & UV index
- 🕐 Today's hourly forecast (6 time slots)
- 📅 7-day daily forecast with high/low temps
- 💨 Air conditions panel (rain chance, humidity, wind, UV)
- 🎨 Dark UI with smooth animations
- ⚡ Single API call — fast and efficient

## 🗂️ Project Structure

weather-dashboard/
├── index.html     # App shell — no hardcoded content
├── style.css      # All styles & CSS variables
└── script.js      # Data, API calls & DOM rendering

## 🚀 Getting Started

1. Clone the repo
   git clone https://github.com/your-username/weather-dashboard.git

2. Get a free API key at https://www.weatherapi.com

3. Open script.js and add your key at the top:
   const API_KEY = 'your_key_here';

4. Open index.html in your browser — no build step needed!
